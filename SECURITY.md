# Security

This repository is **documentation and sample text** only. It does not ship executable services.

## Reporting a security issue

- If you find **sensitive data accidentally committed** (keys, tokens, personal paths), open a issue describing the file and approximate location, or contact the repository maintainers privately if GitHub **private vulnerability reporting** is enabled for this repo.
- For **vulnerabilities in Reticulum itself** or official applications, follow the security process documented by those upstream projects (this notes repo is not the primary security contact for Reticulum).

## Samples and privacy

CLI captures may contain cryptographic hashes or paths. Contributors should redact before publishing; if you spot a leak, report it so it can be removed or rotated in documentation.
