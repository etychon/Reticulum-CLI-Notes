# Config snippets

Only **redacted** or **dummy** interface blocks belong here. Do not commit live destinations, keys, or callsign-specific RF plans.

Files use illustrative names (e.g. `example-tcp-client.toml`, `rnode-lora-interface-eu868-example.txt`). Operators must verify frequency, bandwidth, spreading factor, and transmit power against **local regulations** and hardware capabilities.

For firmware and `rnodeconf` steps, see [docs/linux/rnode-lora-install.md](../../docs/linux/rnode-lora-install.md).
